# Place your project images here

Add `.jpg`, `.png`, `.gif`, or `.webp` files in this folder.

For example:
- `project-1.jpg` - First project image
- `project-2.png` - Second project image
- `project-3.webp` - Third project image

Then reference them in:
1. `projects.json` - Gallery thumbnail images
2. `projects/project-X.html` - Hero/showcase images
