Self-developed hardware project

BMO (Better Macro Operator)

A fully self-developed and self-built macropad created as a practical desktop tool, with several revisions across enclosure design, electronics, and final assembly.

BMO was developed by Dalvin and Amir as a custom desk tool that reflects the way they work. It combines 3D design, coding, electronics, and fabrication into one compact device, built to be useful long after the assignment is finished.

Why it exists

A custom tool that is personal, open, and affordable.

Commercial macropads can be expensive, closed off, and visually generic. BMO was built to solve that by giving its makers full control over the hardware, code, case, and layout. The result is a customizable device that can adapt to gaming, editing, and programming profiles while still looking good on the desk.

Core features

BMO combines flexible controls with clear visual feedback.

The macropad is built around a Raspberry Pi Pico running KMK firmware. It uses a customizable input layout, profile switching, an OLED display for useful information, magnetic connectors for future expansion, and a wooden outer case that gives the project a warmer, more finished presence.

Under the hood, the build includes 16 MX-style switches, 12 diodes, 2 rotary encoders, 16 white LEDs, and the electronics needed to keep the device responsive and expandable.

Version 1 to 3

The first builds focused on making the idea work.

Version 1 started on a breadboard to confirm the components communicated correctly. Version 2 moved into a soldered build with 16 buttons, 12 diodes, and 8 wires. It was fragile, but it proved the concept. Version 3 introduced a rotary encoder and potentiometer on a sturdier protoboard and marked the moment profile switching began to work properly.

Version 4 and 5

The design then moved toward the final appearance.

Version 4 replaced the basic buttons with MX-style switches, added keycaps, and introduced a designed case. The OLED display was tested here too, although it remained a challenge. Version 5 became a true mockup of the final layout, with a full bottom plate that previewed the final size and fit. That version also gave the project its identity and the name BMO.

The final version takes the project further with a fully designed PCB, an angled slope for ergonomics, and light-up switches. The result is much closer to a polished product than a prototype.

Technical documentation

The build is simple in concept, but carefully planned in practice.

The electrical design uses a switch matrix so the controller can read 16 keys with fewer pins. Diodes isolate each switch and prevent ghosting. Assembly starts with the baseplate and brass inserts, then the feet, internal components, PCB, keycaps, knobs, and the wooden cover. The PDF also documents the soldering workflow and the final checks for LEDs, wiring, and input response.

Result

A functional macropad that feels personal and complete.

BMO ended up as a practical desktop tool that blends usefulness with craft. It is customizable, expandable, open enough for others to remix, and finished with the kind of material contrast that makes the project stand out. More importantly, it is something the makers can keep using every day.
